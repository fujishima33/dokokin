# どこ勤  
スマホ・PCで場所を問わず、どこでも勤怠管理ができるWEBアプリです。

## 機能一覧
* マルチ認証
    * 管理者登録
    * 一般ユーザーのCRUD
* 日報・案件・シフト管理データのCRUD
* CSVファイルダウンロード機能
* ページネーション機能
* 画像投稿機能
* カレンダーを利用したデータの参照
* 現在時刻のリアルタイム表示機能
* レスポンシブ対応

## 使用技術
* PHP 7.2
* Laravel 5.8
* javascript
* Mysql 5.7
* Heroku(インフラ)
* AWS
    * cloud9（開発環境）
    * S3
